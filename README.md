# KAHLT and Grouped FFN Transformers

Official implementation of KAHLT and Grouped FFN based transformers.

## Contents

This repository includes implementations for:
- KAHLT (KAN-based Hybrid Latent Transformers)
- Grouped FFN based transformers
- MLP baseline models
- KAN-based models

## Files

The repository contains training scripts for different model architectures on AG News and ArXiv classification datasets.
